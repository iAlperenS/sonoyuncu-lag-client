# SonOyuncu Lag-Client
SonOyuncu Lag Client With Proxy

## Kurulum
- 1. SonOyuncuyu açın sunucu ekleden 127.0.0.1:25565 ekleyin
- 2. main.py yi açın oyuna girin eğer bypass işe yaramazsa kicked whitelist derse
  - C:\Windows\System32\drivers\etc\hosts Yönetici olarak not defterini açın bunu en aşşağı yapıştırın
  ```bash
  127.0.0.1 mc.sonoyuncu.network
  127.0.0.1 sonoyuncu.network
  ```
  - Şimdi normal serverdan deneyin çalışacaktır

## Gerekenler
- Python versionum: 3.13.6

pip3 install keyboard==0.13.5
